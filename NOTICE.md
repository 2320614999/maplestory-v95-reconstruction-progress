## English

This repository is a **public progress display** for the private `maplestory-v95-reconstruction` workspace.

This repository does not distribute, host, or mirror:

- MapleStory client binaries;
- WZ files;
- IDA databases, PDB files, or full decompiler exports;
- runtime logs, dumps, or raw screenshots;
- proprietary Nexon / Wizet assets;
- upstream Kinoko / kinoko_client source trees;
- unknown third-party server packs or localization packs.

This repository only publishes curated non-sensitive progress notes, roadmap material, status tables, and public project descriptions.

### Third-party projects

Kinoko and kinoko_client are related upstream projects. This progress repository only references them as external project context. It does not redistribute their source code and does not relicense upstream code on behalf of their authors.

### Trademarks and assets

MapleStory, Nexon, Wizet, and related names, images, resources, and trademarks belong to their respective owners. This project is for research, reconstruction progress tracking, and engineering documentation.
